# Bienvenido a la libreria de componentes de Kuspit :)

Esta libreria de componentes se hizo con la finalidad de tener documentada la parte de los componentes que se reutilizaran a lo largo de esta aplicación :).
Tiene la finalidad de poder orientar a nuevos desarrolladores que puedan entrar al equipo asi como dar una preview de cada componente utilizado junto con cada uno de los posibles estados de este componente.

Esta documentación fue posible a Storybook y es realmente sencillo implementarla en un entorno de desarrollo.
Creemos que tener una documentación es clave para agilizar el desarrollo :)
